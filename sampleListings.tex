\section{Sample Java listings\label{sec:sampleListings}}
A collection of Java listings \MyTableReference{tbl:startingPoint} that might be of use when designing your CRM implementation.  The files are available on \texttt{GitHub}, and can be cloned to a local repository with this git command:
\begin{center}
\begin{verbatim}
  git clone git@github.com:Chuck-Cartledge/TCC-223-Control-file.git
\end{verbatim}
\end{center}
or via a git graphical user interface \textbf{GUI}.
