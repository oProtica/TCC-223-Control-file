\section{Discussion}
The purpose of this project is to introduce and work with various real-world tools in a safe and instructive manner.  The project will be to create the back end of a persistent client relationship management (\textbf{CRM}) system.  For our purposes:
\begin{itemize}
\item The application software will be written in Java,
\item The CRM database will be persistent between program executions,
\item A GitHub repository will serve as the base application,
  \item Program teams will make team specific branches from the GitHub base,
  \item Team members will use local git repositories for local and team versions,
    \item It will be assumed that a totally different application will interact with the user to create the control data file that is used to give commands to the CRM.
\end{itemize}

Other details will be fleshed out as the semester progresses.
